  • BMViewController

    Overview

    A BMViewController replaces the default UIViewController. It manages a single screen, and listen to the users interactions with it.

    Refer to BMViewWithViewControllerAndViewModel to understand the architecture of a view.

    The BMViewController gives you access to a rootView which is the view properly casted.

  • PDFViewController

    A PDFViewController is UIViewController with an integrated PDFView inside of it. It is assumed that this ViewController will always be inside a navigation controller and therefore the dismissButtonItem is set as the leftBarButtonItem of the navigationItem. The dismissButtonItem is customisable.
